Arduino on the web

You want to put your aduino on the web without the ethernet shield. This project is for you. The goal is to demonstrate how use arduino with python and tornado.

Installation

Clone the depot. Install virtualenv and follow this instruction. 

$ virtualenv init .venv
$ source .venv/bin/activate
$ pip install pyserial tornado
$ python serverled.py --serial /port/serial/arduino

The example code of arduino is in tornado/skeetchbook.
